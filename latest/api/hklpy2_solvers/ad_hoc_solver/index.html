hklpy2_solvers.ad_hoc_solver#

Solver wrapping ad_hoc_diffractometer for hklpy2.

Provides the AdHocSolver class, which implements the hklpy2.backends.base.SolverBase interface on top of the ad_hoc_diffractometer library. All geometries registered with the library are available.

Solver wrapping ad_hoc_diffractometer for hklpy2.

Wraps ad_hoc_diffractometer.diffractometer.AdHocDiffractometer behind the SolverBase interface so that hklpy2 can use it for forward / inverse calculations.

All geometries registered with the ad_hoc_diffractometer library are supported. Geometry discovery is dynamic: any geometry added to the library’s registry (including via entry points) is automatically available.

See also

ad_hoc_diffractometer.list_geometries()

PARAMETERS

geometrystr

Name of the diffractometer geometry. Must be one of the names returned by geometries(). Default: "fourcv".

kwargs

Passed to the geometry factory function. For kappa geometries, use kappa_alpha_deg=... to set the kappa tilt angle.

calculate_UB(r1: hklpy2.backends.typing.ReflectionDict, r2: hklpy2.backends.typing.ReflectionDict) → hklpy2.typing.Matrix3x3[source]#

Calculate the UB matrix using two reflections (Busing & Levy).

The r1 and r2 arguments are the contractual source of truth: any prior reflections held by the underlying ad_hoc_diffractometer sample are cleared and the two named reflections are inserted (so setor0 / setor1 are correctly designated) before UB is computed. See 56.

property extra_axis_names: list[str]#

Ordered list of input extra parameter names for the current mode.

Drawn from the underlying mode’s extras dict (filtered to the names the user is expected to supply: n_hat, psi, alpha_i, beta_out, h2, k2, l2) plus the scalar name of the active ReferenceConstraint if it names one of those inputs and is not already listed.

Solver-output placeholders (e.g. psi populated by the solver after forward()) are also exposed as inputs so the user can change the target value before the next forward call.

Returns [] for modes with no extras.
